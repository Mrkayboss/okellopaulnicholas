## 🎨 Design Features

- **Animated Background**: Continuous gradient shift (15s loop)
- **Floating Elements**: Pseudo-element orbs with depth
- **Smooth Animations**: AOS library for scroll effects
- **Glassmorphic Navbar**: Sticky navigation with blur effect
- **Responsive Design**: Mobile-first approach (breakpoint: 768px)
- **Modern Typography**: Inter font family (weights 300-800)
- **Color System**: Primary blue (#1e3a8a), Secondary blue (#3b82f6)

## 📱 Responsive Breakpoints

- **Desktop**: 1200px max-width container
- **Tablet**: Adjusted layouts, reduced spacing
- **Mobile**: <768px - Single column, optimized spacing
